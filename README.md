# sixth_sense website

Open `index.html` in a browser to view the site.

## Photos

- The real studio photo is used from `assets/portfolio/studio-interior.webp`.
- The tattoo photos were copied into `assets/portfolio/` with readable filenames and resized for web use.
- The portfolio is rendered from the `portfolioItems` list in `script.js`.
- Gallery filters: Featured, Realism, Mythology, Lettering, Fine line, and All.
